## 二、核心模块说明

### 2.1 配置模块 (config.js)

**功能描述**: 存储游戏所有可配置参数和常量

**主要配置项**:
- `COLORS`: 主题色彩配置
- `CARD`: 卡牌尺寸和动画参数
- `CHIP`: 筹码配置
- `GAME`: 游戏规则参数
- `AI`: AI难度配置
- `PATHS`: 资源路径配置
- `AUDIO`: 音频配置
- `STORAGE`: 存储配置
- `ANIMATION`: 动画参数

**使用方式**:
```javascript
const cardWidth = CONFIG.CARD.WIDTH;
const primaryColor = CONFIG.COLORS.PRIMARY_1;
```

### 2.2 卡牌系统 (cardSystem.js)

**类名**: `CardSystem`

**主要功能**:
- 创建和管理52张标准牌组
- 洗牌和发牌
- 计算手牌点数
- 检测Blackjack和爆牌
- 创建卡牌DOM元素

**核心方法**:

| 方法名 | 参数 | 返回值 | 描述 |
|--------|------|--------|------|
| `createDeck()` | 无 | Array | 创建标准52张牌组 |
| `shuffleDeck()` | 无 | void | 洗牌 |
| `dealCard()` | 无 | Object | 发一张牌 |
| `calculatePoints(cards)` | Array | number | 计算手牌点数 |
| `isBlackjack(cards)` | Array | boolean | 检测Blackjack |
| `isBust(cards)` | Array | boolean | 检测爆牌 |
| `createCardElement(card, faceDown)` | Object, boolean | HTMLElement | 创建卡牌DOM元素 |

**卡牌对象结构**:
```javascript
{
    suit: 'guitar',      // 花色
    value: 'A',          // 面值
    point: 11,           // 点数
    isAce: true,         // 是否为A牌
    isFaceCard: false,   // 是否为花牌
    character: {         // 角色信息
        name: '户山香澄',
        role: 'vocalist'
    },
    id: 'unique_id'      // 唯一标识
}
```

### 2.3 AI庄家系统 (aiDealer.js)

**类名**: `AIDealer`

**主要功能**:
- 管理庄家手牌
- 根据难度策略做出决策
- 检测软17

**难度策略**:

| 难度 | 策略描述 | 决策延迟 |
|------|----------|----------|
| 简单 | <17必须要牌，≥17必须停牌 | 1.5秒 |
| 普通 | 17-18有30%概率要牌，软17有50%概率要牌 | 1.0秒 |
| 困难 | 17-19有60%概率要牌，软17有80%概率要牌 | 0.8秒 |

**核心方法**:

| 方法名 | 参数 | 返回值 | 描述 |
|--------|------|--------|------|
| `makeDecision(playerPoints, callback)` | number, Function | void | 做出决策 |
| `isSoft17()` | 无 | boolean | 检测软17 |
| `setDifficulty(difficulty)` | string | void | 设置难度 |

### 2.4 经济系统 (economySystem.js)

**类名**: `EconomySystem`

**主要功能**:
- 管理玩家筹码
- 处理下注和结算
- 统计游戏数据

**核心方法**:

| 方法名 | 参数 | 返回值 | 描述 |
|--------|------|--------|------|
| `placeBet(amount)` | number | Object | 下注 |
| `doubleDown()` | 无 | Object | 加倍下注 |
| `win(isBlackjack)` | boolean | Object | 胜利结算 |
| `lose()` | 无 | Object | 失败结算 |
| `draw()` | 无 | Object | 平局结算 |
| `getStatistics()` | 无 | Object | 获取统计数据 |

**结算规则**:
- 普通胜利: 2倍下注
- Blackjack: 2.5倍下注
- 平局: 返还下注

### 2.5 UI系统 (uiSystem.js)

**类名**: `UISystem`

**主要功能**:
- 创建游戏界面

- 处理动画效果
- 显示弹窗和消息

**核心方法**:

|--------|------|--------|------|
| `init()` | 无 | void | 初始化UI |
| `createGameLayout()` | 无 | void | 创建游戏布局 |

| `flipCard(index, target, callback)` | number, string, Function | void | 翻转卡牌 |
| `showMessage(message, type)` | string, string | void | 显示消息 |

### 2.6 音效系统 (audioSystem.js)

**类名**: `AudioSystem`

**主要功能**:

- 处理静音状态

**可用音效**:
- `deal`: 发牌音效
- `flip`: 翻牌音效
- `chip`: 筹码音效
- `hit`: 要牌按钮音效
- `stand`: 停牌按钮音效
- `double`: 加倍按钮音效
- `win`: 胜利音效
- `lose`: 失败音效
- `bust`: 爆牌音效

**类名**: `SaveSystem`

**主要功能**:

**类名**: `GameController`

**主要功能**:

- 处理玩家操作

**游戏状态**:
- `idle`: 空闲状态
- `betting`: 下注阶段
- `playing`: 游戏进行中
- `ended`: 游戏结束

**核心方法**:

| 方法名 | 描述 |
|--------|------|

| `startGame()` | 开始新游戏 |
| `dealInitialCards()` | 发初始牌 |
| `playerHit()` | 玩家要牌 |
| `playerStand()` | 玩家停牌 |

## 三、游戏流程

### 3.1 游戏状态流程图

```
[初始化] → [下注阶段] → [发牌] → [玩家回合] → [庄家回合] → [结算] → [下注阶段]
     ↓            ↓          ↓          ↓           ↓          ↓
  [加载存档]   [选择筹码]  [检测BJ]  [爆牌检测]  [AI决策]   [显示结果]
```

### 3.2 详细流程

1. **初始化阶段**
   - 加载配置
   - 初始化各子系统
   - 加载存档
   - 显示新手引导（首次游戏）

2. **下注阶段**
   - 显示下注区域
   - 玩家选择筹码
   - 确认下注

3. **发牌阶段**
   - 玩家获得2张明牌
   - 庄家获得1张明牌+1张暗牌
   - 检测初始Blackjack

4. **玩家回合**
   - 选择操作：要牌/停牌/加倍
   - 检测爆牌
   - 点数达到21自动停牌

5. **庄家回合**
   - 翻开暗牌
   - AI根据策略决策
   - 检测爆牌

6. **结算阶段**
   - 比较点数判定胜负
   - 更新筹码
   - 显示结果弹窗
   - 自动保存

## 四、API接口

### 4.1 全局对象

游戏初始化后，可通过 `window.game` 访问游戏实例：

```javascript
// 获取当前筹码
game.economySystem.getChips();

// 设置难度
game.setDifficulty('hard');

// 保存游戏
game.saveGame();

// 获取统计数据
game.economySystem.getStatistics();
```

### 4.2 自定义事件

| 事件名 | 数据 | 描述 |
|--------|------|------|
| `chipSelected` | `{ value: number }` | 筹码被选中 |

**监听事件示例**:
```javascript
document.addEventListener('chipSelected', (e) => {
    console.log('选中筹码:', e.detail.value);
});
```

## 五、扩展指南

### 5.1 添加新角色

1. 在 `assets/art/cards/faces/` 下添加角色立绘
2. 在 `cardSystem.js` 的 `getAvailableCharacterCards()` 方法中添加卡牌文件名
3. 在 `getCharacterForCard()` 方法中配置角色映射

### 5.2 添加新难度

1. 在 `config.js` 的 `AI` 配置中添加新难度
2. 在 `aiDealer.js` 中实现对应的决策策略

### 5.3 添加新音效

1. 在 `audioSystem.js` 的 `createSynthSounds()` 方法中添加新音效
2. 使用 `createTone()` 创建单音或 `createMelody()` 创建旋律
